# Routing

Public routes:

- `/`
- `/services`
- `/request/:serviceSlug`
- `/request-confirmation/:reference`
- `/track/:reference`

Admin routes:

- `/admin`
- `/admin/requests`
- `/admin/requests/:requestId`
- `/admin/suppliers`
- `/admin/services`
- `/admin/analytics`

`AdminRouteGuard` is intentionally a placeholder until authentication exists.
