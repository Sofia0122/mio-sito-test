# Linee Guida UI

Harbourly e mobile-first per i flussi cliente e ottimizzato desktop per le operazioni admin.

Usa Tailwind CSS, componenti UI generici riutilizzabili, label accessibili, focus state e controlli utilizzabili da tastiera. Evita codice business-specific in `components/ui`.

Quando sara disponibile Figma MCP, usa Figma come fonte principale per token, layout, typography e comportamento responsive.
